## 校园公共服务系统项目
* 该项目主要以 `提供数字化校园服务` 为目标，在为游客提供纵览校园服务的同时为校内管理人员提供便捷的数字化管理服务。
#### 项目涉及技术
* `HTML` `CSS` `JS` `jQuery` `Bootstrap` `OpenLayers` `Echart` `Tabulator` `ASP.NET MVC` `C#` `SQL Server`
